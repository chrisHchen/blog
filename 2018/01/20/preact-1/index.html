preact 源码(1)

Preact 是一个只有 3kb 的 react 替代库。Preact 有 90% 的 API 是和 react 兼容的。所以用 preact 来开发对于熟悉 react 的人来说是非常方便的。

Preact 整个项目是用 rollup 来构建，构建的配置文件在 config/rollup.config.js, 入口文件是 src/preact.js

jsx 与 createElement/h
看 createElement 函数前有必要先了解下 jsx。 这篇文章(wtf-is-jsx)非常好的解释了什么是 jsx 以及 jsx 如何转化为 vnode!

比如 <div id="foo" name="bar">Hello!</div> 这样的 jsx 通过 babel 转义后会产生 h('div', { id: 'foo', name : 'bar' }, 'Hello!'); 这样的代码，其中的 createElement 也可以写成 h，意思是一样的。createElement 函数的执行后返回的就是 VNode，也就是常说的虚拟 dom 节点。

createElement(nodeName, attributes, …rest) ，文件：scr/h.js
第一个参数是 string 类型的 nodeName ，比如 ‘div’, ‘span’
第二个参数是 object 类型的 attributes，也就是属性集合。
从第三个参数以及之后的参数都被认为是 children

从代码中可以看出：createElement 函数会对 children 做扁平化处理，也就是传入的 children 可以是任意嵌套的数组。
另外，如果没有传入第三个参数但第二个参数对象有 children 这个属性，那么 children 属性会被作为 children 来对待。
如果传入了第三个参数，则第二个参数对象里的 children 属性会被忽略。
